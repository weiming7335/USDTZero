# USDTZero

一个支持多链USDT支付的服务，支持TRC20、SPL、BEP20等主流区块链网络。

## 功能特性

- 🔗 **多链支持**：目前支持TRC20、SPL、BEP20
- 🔄 **实时监听**：区块链交易实时监控
- 📡 **回调通知**：支付状态回调
- 📊 **监控指标**：应用性能监控
- 🤖 **Telegram Bot**：交易消息推送
